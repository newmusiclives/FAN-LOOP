Trigger Configuration

  • Trigger Type: Contact Changed > Custom Field Updated
  • Field: referral_count
  • Allow Re-entry: Yes (must fire on every update)

Step-by-Step GHL Setup

  1. Create a new workflow. Add trigger: Contact Changed. Select the custom field referral_count.
  2. Add an If/Else branch. Set condition: referral_count equals 3. In the "Yes" path, add a Send SMS action with the 3-referral message.
  3. In the "No" path, add another If/Else branch. Condition: referral_count equals 5. Add the 5-referral SMS in the "Yes" path.
  4. Continue nesting: add another If/Else for referral_count equals 10. Add the 10-referral SMS.
  5. Final If/Else: referral_count equals 25. Add the 25-referral SMS.
  6. Publish the workflow.

Trigger Configuration

  • Trigger Type: Contact Tag Added
  • Tag: fanloop
  • Initial Wait: 7 days after trigger
  • Entry Condition: Custom field referral_count equals 0
  • Run Once Per Contact: Yes

Step-by-Step GHL Setup

  1. Create a new workflow. Add trigger: Contact Tag Added with tag fanloop.
  2. Add a Wait step set to 7 days.
  3. Add an If/Else condition: referral_count equals 0. Only proceed on "Yes" path — the "No" path means they've already referred someone, so we end the workflow.
  4. In the "Yes" path, add Send SMS with the Day 7 message below.
  5. Add a Wait step set to 3 days.
  6. Add another If/Else condition: referral_count still equals 0. This re-checks in case they referred someone after the Day 7 SMS.
  7. In the "Yes" path, add Send Email with the Day 10 email below.
  8. Add a Wait step set to 4 days.
  9. Add a final If/Else: referral_count equals 0.
  10. In the "Yes" path, add Send SMS with the Day 14 final nudge below.
  11. Publish the workflow.

Pre-Requisite: Create the VIP Pipeline

Before building this workflow, create a pipeline in GHL:

  1. Go to Opportunities > Pipelines and click + Create Pipeline.
  2. Name it "VIP Fans".
  3. Create these stages: New VIP → Outreach Scheduled → Contacted → Rewards Delivered → Ambassador Active.
  4. Save the pipeline.

Step-by-Step GHL Setup

  1. Create a new workflow. Add trigger: Contact Changed on field referral_count.
  2. Add an If/Else condition: referral_count is greater than or equal to 25.
  3. In the "Yes" path, add Add to Pipeline action. Select the "VIP Fans" pipeline, stage "New VIP".
  4. Add Add Tag action. Tag: vip_fan.
  5. Add Internal Notification action. Set to email your admin. Use the admin notification template below.
  6. Add Send SMS action with the VIP welcome message below.
  7. Add a Wait step set to 10 minutes.
  8. Add Create Task action. Title: "Personal outreach to VIP fan: {{contact.first_name}} {{contact.last_name}}". Due date: 1 day from now. Assign to your team lead.
  9. Add Update Pipeline Stage action. Move opportunity to "Outreach Scheduled" stage.
  10. Publish the workflow.

Custom Field Setup Guide

These custom fields must be created in GHL before the workflows will function. Here is every field you need, its type, and its purpose.

Required Custom Fields

Navigate to Settings > Custom Fields in GHL and create each of the following:

Field Name Field Key Type Purpose
Referral Count referral_count Number Total referrals this contact has generated. Updated by TrueFans LOOP webhook.
Referral Link referral_link Text The contact's unique referral URL. Populated on signup.
Referral Tier referral_tier Dropdown Current tier: None, Tier 1, Tier 2, Tier 3, VIP
Campaign ID campaign_id Text The TrueFans LOOP campaign this contact belongs to.
Referred By referred_by Text Email or ID of the contact who referred this person.
Signup Date loop_signup_date Date When the contact joined the campaign via TrueFans LOOP.
Last Referral Date last_referral_date Date Timestamp of the most recent referral. Used for re-engagement logic.
Rewards Claimed rewards_claimed Text (Multi-line) Comma-separated list of claimed reward IDs. Prevents duplicate fulfillment.
Shipping Address shipping_address Text (Multi-line) For physical rewards. Collected when fan hits a physical reward tier.
VIP Status vip_status Dropdown Options: Not VIP, VIP Pending, VIP Active, Ambassador

Required Tags

Create these tags in GHL (Settings > Tags or create them on-the-fly):

Tag Applied When
fanloop Contact joins any TrueFans LOOP campaign
tier_1_unlocked Contact hits 3 referrals
tier_2_unlocked Contact hits 5 referrals
tier_3_unlocked Contact hits 10 referrals
tier_4_vip Contact hits 25 referrals
vip_fan Contact moves into VIP pipeline
re_engagement_sent Re-engagement workflow has started for this contact
Important: Field keys must match exactly. TrueFans LOOP webhooks reference these keys when updating contact data. A mismatch means the workflows won't trigger properly.

Import & Setup Instructions

Follow this step-by-step guide to recreate all five workflows in your GoHighLevel account.

Complete Setup Checklist

  1. Create Custom Fields
    Go to Settings > Custom Fields. Create all 10 fields listed in Section 7. Double-check that the field keys match exactly.
  2. Create Tags
    Go to Settings > Tags (or create them when first used in a workflow). Create all 7 tags listed in Section 7.
  3. Create the VIP Pipeline
    Go to Opportunities > Pipelines. Create the "VIP Fans" pipeline with stages: New VIP → Outreach Scheduled → Contacted → Rewards Delivered → Ambassador Active.
  4. Build Workflow 1: New Fan Welcome
    Go to Automation > Workflows > Create Workflow. Follow the step-by-step instructions in Section 2. Copy-paste all message content from this page using the Copy buttons.
  5. Build Workflow 2: Referral Milestone
    Create a new workflow. Follow Section 3 instructions. Set up all 4 If/Else branches for milestones 3, 5, 10, and 25.
  6. Build Workflow 3: Reward Fulfillment
    Create a new workflow. Follow Section 4 instructions. Set up the tier tag triggers and all 4 reward emails.
  7. Build Workflow 4: Re-engagement
    Create a new workflow. Follow Section 5 instructions. Make sure the 7-day initial wait is configured correctly.
  8. Build Workflow 5: VIP Pipeline
    Create a new workflow. Follow Section 6 instructions. Make sure the VIP Fans pipeline exists before building this workflow.
  9. Configure TrueFans LOOP Webhooks
    In your TrueFans LOOP campaign settings, add a webhook URL pointing to your GHL account's inbound webhook endpoint. Map the fields: email, first_name, last_name, phone, referral_count, referral_link, campaign_id.
  10. Test the Integration
    Create a test contact in TrueFans LOOP. Verify the webhook fires, the contact appears in GHL with the correct custom fields, and the New Fan Welcome workflow triggers. Manually update the referral_count field to test milestone and VIP workflows.
  11. Publish All Workflows
    Toggle each workflow from "Draft" to "Published." Double-check trigger conditions and If/Else logic. Monitor the first few contacts to ensure everything fires correctly.
  12. Customize Message Content
    Replace all [ARTIST NAME], [SONG TITLE], and placeholder links with your actual content. Update reward descriptions to match your specific tier offerings. Adjust timing delays if needed for your audience.
Estimated Setup Time: 45-60 minutes for all five workflows if you have the custom fields and pipeline already created. Budget an additional 15 minutes for webhook configuration and testing.
